## 3. Quick Start

1. Install Rails 7
  ```bash
  gem install rails
  ```

2. Clone
  ```bash
  git clone git@github.com:ThibautBaissac/search_filters.git
  ```

3. Bundle & yarn install
  ```bash
  bundle
  yarn install
  ```

4. Run rails db commands
  ```bash
  rails db:create
  rails db:migrate
  rails db:seed
  ```

5. Run rails server
  ```bash
  ./bin/dev
  ```
